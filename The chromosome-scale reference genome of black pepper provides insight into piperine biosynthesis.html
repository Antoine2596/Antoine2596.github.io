Présentation la figure et des données

Présentation du circos

Il s'agit d'un circos constitué de 12 tracks et d'un centre constitué de lien entre différents gène du génome de référence du poivre noir (Piper nigrum).

La piste b

La piste (b) représente la distribution de la densité GC sous forme d'histogramme.

La piste c

La piste (c) représente la densité de répétition sous forme d'un tracé de trait.

La piste d

La piste (d) représente la densité de gènes sous forme d'histogramme

La piste e,f,g,h

Les pistes (e), (f), (g) et (g) représente l'expression de gènes spécifiques à la baie sous forme d'histogramme.

La piste i,j,k,l

Les pistes (i), (j), (k) et (l) représente le niveau de transcription sous forme de nuage de points.

La piste m

La piste (m) reflète les emplacements des gènes cartographiés sur les voies du métabolisme secondaire et du métabolisme des alcaloïdes.

La piste n

La piste (n) représente les blocs synténiques (présence simultanée sur le même chromosome de deux ou plusieurs locus).

Présentation des données

Toutes les données nécéssaires à la réalisation de ce circos se situent dans le fichier 41467_2019_12607_MOESM4_ESM.xlsx.

A l'intérieur, toutes plupart des données des pistes sont structuré de façon à être interprétable par la librarie 'BioCircos'.

Cependant le tableau pour la réalisation du track (n), c'est à dire celui des liens, ne peut pas être interprété.

Il faudra ici mettre les données possèdant le même nom "links" sur la même ligne.

Remaniement des données pour la piste (n)

On créer deux tableaux, un contenant les lignes pairs, et un autre contenant les lignes impaires. On fait ensuite une fusion des deux tableau à partir de leur colonne commune "links".

# Filtrer les indices de lignes impairs
indices_impairs <- seq(1, nrow(FigN), by = 2)

# Supprimer les lignes impaires du tableau
ChrA <- FigN[indices_impairs, ]

# Filtrer les indices de lignes pairs
indices_pairs <- seq(2, nrow(FigN), by = 1)

# Supprimer les lignes paires du tableau
ChrB <- FigN[indices_pairs, ]

#On fait une fusion des tableaux à partir de la colonne commune "Links"
FigN <- merge(ChrA, ChrB, by = "Links")

Affichage du titre

Biocircos est un package qui à utilise des fonctions différentes selon le type de données qu'on souhaite afficher (heatmap, link, line,...), un circos a donc besoin d'une addition de chaque fonctions afin de contenir toutes les données.

Pour commencer, nous allons définir un objet nommé 'Track' qui en premier temps contenir la fonction 'BioCircosTextTrack', afin d'afficher un texte. Ici, nous allons afficher le titre de la figure.

#Créer un titre
Track <- BioCircosTextTrack("MyTextTrack",#Nom sans importance qu'on donne pour nommé le rôle de cette fonction
"Genome of black pepper", #Le texte à afficher
x = -0.4 # Position en abscisse du texte
,y=-1.5, # Position en ordonnée du texte
size = "1em") #taille du texte

Affichage de la piste n

La piste m est un ensemble de liens interchromosomique, j'ai utiliser une boucle afin de charger chaque ligne avec une couleur différentes aléatoirement afin que chacunes de ces liens puissent être distinguées.

# Créer les noms de lignes
Chromosome <- c(paste0("Pn", 1:26)) #Créer une liste "Chromosome" contenant les Pn de Pn1 à Pn26
table_couleurs <- as.data.frame(Chromosome) #On fait un tableau à partir de la liste

#Ajout des couleurs
for(i in 1:26){
couleur_aleatoire <- paste0("#", paste(sample(c(0:9, letters[1:6]), 6, replace = TRUE), collapse = "")) #On créer une couleur aléaloitre en hexadécimal
table_couleurs$couleurs[i] <- couleur_aleatoire #On place cette couleur dans une case du tableau
table_couleurs <- rbind(table_couleurs, couleur_aleatoire) #combiner les tableaux
}

#On renomme les lignes par les noms des chromosomes
rownames(table_couleurs) <- table_couleurs$Chromosome

#On doit au préalable avoir une variable définit avant de lancer la boucle, peut importe combien elle vaut.
NomPn<-1
# Dans ce script, j'ai réaliser un tableau de couleur avec une couleur pour chaque PN, ainsi les liens auront des couleurs en fonction du chromosome de départ
# On créer une boucle qui va chargée les liens une par une afin d'avoir leur couleurs correspondante
for (i in 1:2589){
Track <- Track + BioCircosLinkTrack("FigN",
FigN$Chromosome.x[i],
FigN$Start.x[i],
FigN$End.x[i],
FigN$Chromosome.y[i],
FigN$Start.y[i],
FigN$End.y[i],
maxRadius = 0.4,
NomPn <- FigN$Chromosome.x[i],
color = table_couleurs[NomPn,2],
displayLabel = FALSE)
}
