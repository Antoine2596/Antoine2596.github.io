I. Présentation du machine learning

Le Maching Learning est une branche de l'Intelligence Artificielle (IA) qui permet à une machine d'apprendre automatiquement à partir de données. Le Maching Learning se base sur des algorithmes mathématiques, permettant à un ordinateur de reconnaître des tendances (c'est-à-dire des modèles mathématiques responsable des relations dans les données), puis de les utiliser afin de prendre des décision ou de faire des prédictions.
